## 2. 当前 **无法从所提供 PDF 自动抽取** 的内容（须人工补）

以下在正式核对前 **不得**写入 `config/gbt50082_methods.yaml` 的 `formula` 字段为“已验证”：

- 各收缩试验方法中 **应变/标距/温度修正** 的完整公式编号与分母定义。  
- 早期抗裂试验中 **单位面积裂缝数目、最大裂缝宽度统计口径、终止观测条件** 对应公式或条文表。  
- 与 **裂缝宽度 / 裂缝密度 / 开裂风险** 在贵司 `SteelFiberCrackPredictor` 主链路中已有列名的 **一一映射条款**（若标准正文使用不同术语，须在映射表中显式给出对照）。
